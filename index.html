Visit Jax!

This application was built using vanilla JavaScript, jQuery, and Knockout.js. It utilizes Google Maps API, Google Places API, and the Foursquare API to create a map of a neighborhood and populate it with map markers containing information about each venue, a list view, and a search bar to filter the list. I built this application for practice with API calls and the MVVM approach to application building through Knockout.js.

Dankneiss Arts

This website was built for an up and coming artist located in Sarasota, Florida. The layout utilizes a bootstrap template that has been customized to fit the client's needs. The layout was chosen to be a single-page scroll website that was purposefully kept minimalistic in nature so as to not distract from the actual artwork on the site. Also included was a custom made PHP contact form to allow anyone to directly contact the artist himself. Future renovations planned include implementing the PayPal API to allow for artwork to be sold through the site.

Frogger Clone

This project is a simple clone of the popular game frogger. In it, the user controls a player character and has to avoid the moving bugs to reach the water on the other side of the map. This application was built to demonstrate an understanding of object-oriented JavaScript programming and serves as more of a proof of ability.
